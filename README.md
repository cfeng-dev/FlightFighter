# FlightFighter

A simple text-based shooting game developed in C, where players control an airplane to shoot down enemies and score points.

## Prerequisites

You need to have a **C compiler** and **CodeLite IDE** installed on your PC.

## Controls

-   `w`- move up
-   `s`- move down
-   `a`- move left
-   `d`- move right
-   `space`- shoot
